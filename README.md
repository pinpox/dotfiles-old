# Installation

## Dependencies
- git
- ruby
- python
  - python-neovim
- neovim
- zsh
- fzf

## Setup

To setup the dotsfiles on a new system run the setup script using

```
curl -Lks http://bit.do/dotinstall | /bin/bash
```
or
```
curl -Lks https://gitlab.com/binaryplease/dotfiles/raw/master/bootstrap.sh | /bin/bash
```
